## Deskripsi Proyek

Proyek ini bertujuan untuk menganalisis data penyewaan sepeda harian
menggunakan dataset bike sharing. Analisis dilakukan untuk memahami pola
penyewaan sepeda berdasarkan waktu serta pengaruh kondisi cuaca terhadap
jumlah penyewaan.

## Pertanyaan Bisnis

1.  Berapa jumlah rata-rata penyewaan sepeda pada tahun 2011 sampai
    2012?
2.  Seberapa besar pengaruh kondisi cuaca terhadap penyewaan sepeda
    harian pada tahun 2011 sampai 2012?
